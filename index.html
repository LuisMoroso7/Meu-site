Projetos Acadêmicos

Cirífood

Protótipo desenvolvido no Figma para um aplicativo de delivery, com foco em interfaces de baixa fidelidade. O projeto estruturou o fluxo principal do usuário — desde o cadastro até a finalização do pedido — priorizando usabilidade, simplicidade e clareza na experiência visual.

Estrutura de Loja

Projeto acadêmico em Python que simula um sistema simples de estoque e vendas no terminal. A aplicação utiliza estruturas de dados clássicas, como listas, pilhas e filas, integradas ao paradigma de orientação a objetos (POO), demonstrando organização lógica e boas práticas de programação.

Tratamento de Dados

Notebook em Python desenvolvido no Google Colab voltado para análise e experimentação com dados. O projeto demonstra habilidades em manipulação, visualização e exploração de informações, além de aplicar conceitos de aprendizado de máquina em um ambiente interativo e acessível.

Biblioteca Virtual

Aplicação em Java desenvolvida para gerenciar tanto livros físicos quanto digitais. O sistema permite cadastrar, consultar e organizar obras de forma prática em um ambiente virtual, evidenciando conceitos de orientação a objetos e boas práticas de programação estruturada.

Jogos Mortais

Protótipo de jogo em Python que coloca o jogador diante de escolhas interativas, explorando lógica condicional e diferentes caminhos de decisão. Desenvolvido como projeto acadêmico, destaca habilidades em programação estruturada e criação de narrativas simples dentro do console.

Stranger Things Game

Jogo 2D desenvolvido em Python no qual o jogador controla a personagem Onze, da série Stranger Things, no Mundo Invertido. Durante a jornada, é preciso enfrentar desafios enquanto o vilão Vecna tenta capturá-la. O projeto combina controles simples, estética em estilo desenho animado e uma trilha sonora inspirada na série, proporcionando uma experiência imersiva e dinâmica.
